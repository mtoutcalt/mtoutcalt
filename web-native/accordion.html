Animated Height ::details-content

Smooth height animation using the ::details-content pseudo-element:

How does the animation work?
The ::details-content pseudo-element animates from height: 0 to height: auto using the new interpolate-size CSS property. The content-visibility property is also transitioned with the allow-discrete keyword.
What browsers support this?
Chrome 131+, Edge 131+, Firefox 143+, and Safari 18.4+ support the ::details-content pseudo-element. In browsers without support, the accordion still works but without the smooth animations.
Can I animate other properties?
Yes! You can animate width, opacity, transforms, borders, and more. The key is to also transition content-visibility with allow-discrete to handle the show/hide behavior properly.
